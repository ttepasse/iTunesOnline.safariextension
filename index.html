iTunesOnline.safariextension by tobiastom

Prevents iTunes from opening when visiting iTunes Preview pages

Installation

Enable extensions in Safari 5 and download iTunesOnline. Double-click on the file, and Safari will ask you to install this extension.
Once confirmed it should be available inside the extension list, and iTunes should no longer be opend automatically.

There are no settings and no options.

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/tobiastom/iTunesOnline.safariextension

get the source code on GitHub : tobiastom/iTunesOnline.safariextension
